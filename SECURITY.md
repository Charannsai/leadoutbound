# Security Policy

## Supported Versions

Only the latest version of OutReach AI is actively supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do NOT open a public issue. Instead, report it responsibly by contacting the maintainer or community leaders directly.

Please include the following details in your report:
- The type of vulnerability (e.g., SQL injection, XSS, SSRF)
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes or workarounds

We will investigate the issue and address it in a timely manner. Thank you for helping keep this project secure!
